# Final work1
 Контрольная работа
```markdown
Описание задачи

Программа, написанная на C#, решает задачу формирования нового массива из строк, 
длина которых меньше или равна 3 символам. 
Первоначальный массив строк вводится с клавиатуры.

Примеры:
1. Ввод: ["Hello", "2", "world", ":-)"]
   Вывод: ["2", ":-)"]

2. Ввод: ["1234", "1567", "-2", "computer science"]
   Вывод: ["-2"]

3. Ввод: ["Russia", "Denmark", "Kazan"]
   Вывод: []

Алгоритм:

1. Ввод или задание массива строк.
2. Инициализация массива результатов.
3. Для каждой строки в массиве:
   - Если длина строки меньше или равна 3, добавить в массив результатов.
4. Вывести массив результатов.

Инструкция как запустить программу:

1. Склонируйте репозиторий на свой компьютер.
2. Откройте проект в среде разработки C# (например, Visual Studio или Visual Studio Code).
3. Запустите программу.
4. Введите строки массива через пробел при запросе в консоли.
5. Результат выполнения программы будет выведен в консоль.

```
